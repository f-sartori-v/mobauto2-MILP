# mobauto2-milp

Monolithic MILP approach for the MobAuto2 project.

**Quick Start**

From the repository root:

```bash
pip install -e .
mobauto2-milp run
python -m mobauto2_milp run
```

**Package**

- CLI entrypoint: `mobauto2-milp`
- Python module: `mobauto2_milp`
- Config schema: `mobauto2_milp_config`

**Config**

The default configuration lives in [`configs/default.yaml`](/C:/Users/fabio/PyCharm/mobauto2-MILP/configs/default.yaml).

**Notes**

- The solver is a single integrated MILP model.
